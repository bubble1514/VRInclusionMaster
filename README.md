VRInclusionMaster — упрощённая графическая оболочка для scrcpy, предназначенная для быстрого и удобного запуска трансляции изображения с автономных VR-устройств (Pico, Oculus / Quest) на ПК. Приложение ориентировано на использование в образовательной среде и позволяет запускать трансляцию без работы с командной строкой — всего несколькими кнопками.

Проект основан на guiscrcpy, но полностью переработан под практические задачи: удалены избыточные элементы интерфейса, упрощён UI и добавлены функции подключения по Wi-Fi, запуска трансляции в высоком и низком качестве, остановки стрима и установки APK через adb. Решение подходит для менторов и преподавателей без технической подготовки.
